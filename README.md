# Bloomberg Media Platforms

Bloomberg Media Platforms encompass Bloomberg's digital news and content distribution channels including Bloomberg.com, Bloomberg Businessweek, Bloomberg Markets, Bloomberg Technology, Bloomberg Opinion, and Bloomberg Quicktake.

## APIs

- **Bloomberg News API** - Real-time financial news data feeds
- **Bloomberg Media API** - Editorial content distribution API
